# work-template

a template for basic projects. 

## How to use

- Clone the repository to your desktop
- open the folder in your text editor
- Watch css with the command in your terminal "sass --watch scss:css"
- Start working!

## Technologies

- Gitbash
- Text Editor (VScode, Atom, etc.)
- Node
- Sass
